# Automação de Cadastro de Produtos🤖

Projeto desenvolvido em Python para automatizar o cadastro de produtos em um sistema web.

## Tecnologias utilizadas

- Python
- PyAutoGUI
- Pandas

## Funcionalidades

- Abertura automática do navegador
- Login no sistema
- Leitura de dados através de arquivo CSV
- Cadastro automático de produtos
- Preenchimento de formulários

## Como funciona

A automação utiliza uma base de produtos em CSV e realiza o cadastro das informações automaticamente no sistema, reduzindo tarefas repetitivas.

## Arquivos

- `automacao.py` → código principal
- `pegar_posicao_mouse.py` → auxiliar para capturar posições
- `produtos.csv` → base de dados utilizada
